# Нуар UX/UI — візуальна концепція та покращення

Структура гри, екрани та механіка **не змінюються**. Змінюються лише стиль, кольори, типографіка, атмосфера та дрібні UX-елементи.

---

## 1. Візуальна концепція

### Натхнення
- **LA Noire** — темне місто, підсвічені обличчя, поліцейські справи.
- **Papers Please** — папір, штампы, офіційні форми.
- **Disco Elysium** — наратив, дощ, меланхолія.
- **Sherlock Holmes UI** — нитка, докази на дошці, газетний/архівний вигляд.

### Атмосфера
- **Дощ** — опційна легка сітка/overlay (CSS), темний фон.
- **Темне місто** — глибокий синьо-сірий або майже чорний фон, м’яке затемнення.
- **Поліцейські файли** — блоки як папки справ: кремовий/жовтуватий папір, тонкі рамки.
- **Дошка з доказами** — картки підозрюваних як «файли» з червоною ниткою (акцент).
- **Червона нитка** — акцентний колір для важливого: кнопки звинувачення, hover на картках, межі, підсвітка.

### Кольори
| Роль | Опис | Приклад |
|------|------|--------|
| Фон | Темне місто, ніч | `#0d0f12`, `#12151a` |
| Папір / картка | Кремовий, трохи жовтуватий | `#2a2622`, `#352f28` |
| Текст | Кремовий, м’який білий | `#e8e2d9`, `#c9c2b8` |
| Текст другорядний | Приглушений | `#8a847a` |
| Червона нитка (акцент) | Глибокий червоний / бордо | `#8b2942`, `#a62d4a` |
| Підсвітка / hover | Теплий червоний | `#c41e1e`, `rgba(139,41,66,0.4)` |
| Успіх (справа розкрита) | Теплий золотистий | `#c9a227`, `#e8d48b` |
| Помилка (невірно) | Різкий червоний | `#b91c1c` |

### Типографіка
- **Заголовки / логотип** — серіф або «друкарська» гарнітура (наприклад Lora, Crimson Pro), трохи ретро.
- **Основний текст** — читабельний sans (Open Sans або аналог), щоб довгі тексти справи та біографії не втомлювали.
- **Підписи, лейбли** — капс, збільшений letter-spacing, приглушений колір (як у паперах).

## 2. Покращення по екранах

### 1️⃣ Головне меню
- **Фон** — той самий темний + опційно дуже легкий «дощ» (повторювана сітка або градієнт).
- **Назва гри** — серіф, більший розмір, колір «папір» або кремовий.
- **Опис** — той самий блок, трохи менший розмір, letter-spacing для «газетного» відчуття.
- **Вибір складності** — виглядає як **поле з папки справи**: світліший прямокутник (папір), тонка рамка, лейбл зверху (наприклад «Рівень складності» або «Складність справи»). Варіанти — радіо або кнопки-табі в одному стилі.
- **Кнопка «Почати гру»** — головна CTA: червона нитка (фон або рамка), при hover — легка підсвітка або тремтіння (дуже м’яка анімація). Зберігається flip-ефект, але кольори в нуар-палітрі.
- **Індикатор складності** — виключно візуальний: обраний рівень виглядає «обраним» (наприклад товстіша рамка кольору червоної нитки або фон «папір» темніший).

### 2️⃣ Екран справи
- **Верх (опис справи)** — текст у «блоці звіту»: фон як папір (`#2a2622`), рамка, внутрішні відступи. Шрифт читабельний, можливо трохи збільшений line-height. Перший рядок можна трохи виділити (наприклад капс «СПРАВА» або «ЗВІТ» зверху).
- **Центр (картки підозрюваних)** — див. розділ 3.
- **Низ (кнопки)** — «Звинуватити» — акцент червоної нитки, виглядає як критична дія; «Нова справа» — вторинна (outline або приглушений фон). Обидві з тими ж hover-ефектами.
- **Блок «Підозрюваних: N»** — виглядає як **штамп на справі**: прямокутник, серіф-подібний шрифт, рамка «штампа» (наприклад подвійна лінія або кольорова рамка).

### 3️⃣ Екран підозрюваного
- **Контейнер** — один великий «аркуш» папера: кремовий/жовтуватий фон, тінь, ніби документ з справи.
- **Великий портрет** — в рамці (наприклад тонка лінія «фото з справи»), можливо легка сепія або контраст (filter) для нуар-відчуття.
- **Ім’я** — серіф, акцент (наприклад темніший або кольору червоної нитки).
- **Професія** — під ним, меншим, приглушений колір.
- **Біографія** — блок тексту з чіткою типографікою, можливо border-left акцентного кольору для цитат/важливих рядків.
- **Особисті дані** — у вигляді **поля з папера**: лейбли капсом, значення читабельно.
- **Кнопка «Допитати»** — червона нитка, при натисканні можлива коротка мікро-анімація (наприклад затемнення і поява тексту свідчення з fade-in).

---

## 3. Картки підозрюваних

- **Форма** — залишаються ті самі (портрет збоку + список даних). Візуально — як **картка з справи** або **файл підозрюваного**.
- **Фон** — папір: `#2a2622` / `#352f28`, тонка рамка (наприклад `#4a443c` або приглушений колір).
- **Портрет** — в рамці, при hover — легка підсвітка по краю (червона нитка) або дуже м’яка сепія.
- **Ім’я** — виділене: більший розмір, серіф або напівжирний, колір кремовий/білий. Візуально «головна інформація» картки.
- **Докази / дані** — лейбли капсом, letter-spacing, приглушений колір; значення — контрастніше. Розмір взуття, колір волосся тощо виглядають як «запис у справі».
- **Hover** — рамка кольору червоної нитки, легка тінь, можливо дуже невеликий підйом (transform: translateY(-2px)).
- **Клік / відкриття** — без зміни структури: лише візуальний перехід (наприклад картка на мить підсвічується перед переходом на сторінку підозрюваного).

---

## 4. UX-покращення

- **Hover на картках** — рамка + тінь + невеликий підйом; курсор pointer.
- **Hover на кнопках** — підсвітка (box-shadow) кольору акценту, плавний перехід.
- **Індикатор складності** — обраний рівень чітко видно (рамка/фон); не додаємо нових блоків, лише стилізуємо існуючий вибір.
- **«Підозрюваних: N»** — виглядає як штамп (шрифт, рамка), без зміни місця та призначення.
- **Прогрес** — без додавання нових елементів у розмітку; якщо потрібно, можна додати один рядок тексту в існуючий блок зверху (наприклад «Справа · N підозрюваних») без зміни логіки гри.
- **Анімації** — м’які: fade-in при завантаженні блоків (0.2–0.3 s), scale-in для модалок, hover transition 0.2–0.3 s.

---

## 5. Мікровзаємодії

- **Натискання кнопки** — короткий scale (0.98) на active або легке затемнення.
- **Допит** — поява тексту свідчення з fade-in (0.3 s), можливо border-left акцентом.
- **Звинувачення** — модалка з’являється з scale-in; кнопки вибору підозрюваного при hover — підсвітка рамки.
- **Відкриття картки** — перехід на сторінку підозрюваного без зміни; можна додати дуже короткий transition на поточній сторінці (наприклад fade-out контенту).

---

## 6. Атмосфера без зміни механіки

- **Фон** — темний, можливо легкий градієнт (верх світліший, низ темніший) або дуже ненав’язлива сітка «дощу».
- **Контраст** — текст завжди читабельний (WCAG); акцент — для дій і важливих елементів.
- **Папір і картки** — єдиний стиль «справа / папка» на всіх екранах.
- **Червона нитка** — один акцентний колір для hover, CTA, меж, штампів.
- **Тиша** — анімації не відвертають, лише підкреслюють кліки та переходи.

Усе вище реалізується лише зміною CSS (і при потребі шрифтів); структура HTML та компонентів залишається такою ж.
